# **App Name**: Designer Knight Portfolio

## Core Features:

- Hero Section with Parallax: Visually striking hero section with layered dark geometric shapes and parallax scrolling.
- Projects Showcase: Responsive grid layout displaying Muhammad Taha's design projects with hover effects and parallax.
- About Section: Confident bio section with professional avatar, styled with a bold, chivalric tone.
- Contact Form: Clean and minimal contact form with parallax background, integrated with contact details (email, phone, Instagram).

## Style Guidelines:

- Primary color: Crisp Green (#00FF88) for interactive elements and highlights, embodying a modern aesthetic.
- Background color: Dark Gray (#1A1A1A), nearly black, for project cards and section backgrounds, offering strong contrast.
- Accent color: Black (#000000) for primary backgrounds and large text.
- Body text font: 'Inter' (sans-serif) for a clean, modern look.
- Headline font: 'Poppins' (sans-serif) to create a strong visual hierarchy.
- Subtle, knight-themed icons in green for visual accents (e.g., shield, helmet outline).
- 12-column grid system for responsive layout and content structure.
- Smooth parallax scrolling animations with CSS transform: translateY() for dynamic and immersive experience.
